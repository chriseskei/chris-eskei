# Personal Expense Tracker
A simple command-line expense tracker application built with Python.
## Features
- Add expenses with amount, category, and description
- View all expenses in a list
- View summary of expenses by category
- Data persistence using JSON file storage

## How to Use
1. Clone or download this repository
2. Run the application: `python expenses_tracker.py`
3. Follow the menu options to manage your expenses

## Project Structure
- `expenses_tracker.py` - Main application code
- `expenses.json` - Data file (created automatically)
- `requirements.txt` - Project dependencies
